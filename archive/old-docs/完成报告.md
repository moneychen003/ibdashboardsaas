## 📁 关键文件

### 数据文件
| 文件 | 说明 |
|------|------|
| `data/ibkr-YYYYMMDD-HHMMSS.xml` | IB 原始 XML 数据 |
| `data/dashboard_data.json` | 转换后的 JSON（前端使用） |
| `data/kline_data.json` | K 线数据 |
| `data/quotes.json` | 实时行情 |

### 脚本文件
| 脚本 | 功能 |
|------|------|
| `scripts/auto_refresh.sh` | **一键刷新**（推荐） |
| `scripts/ib_to_dashboard.py` | XML → JSON 转换 |
| `scripts/fetch_quotes.py` | 获取实时行情 |
| `scripts/kline_chart.py` | 获取 K 线数据 |

## 🔧 自动化配置

### Cron 定时任务
```bash
crontab -e

# 每个交易日早上 8:30 刷新（美股盘前）
30 8 * * 1-5 /Users/mc/ib_dashboard/scripts/auto_refresh.sh

# 每个交易日晚上 8:30 刷新（美股盘后）
30 20 * * 1-5 /Users/mc/ib_dashboard/scripts/auto_refresh.sh
```

### 服务器自动启动
```bash
# 添加到 ~/.zshrc 或 ~/.bashrc
alias ibdashboard='cd /Users/mc/ib_dashboard && python3 -m http.server 8080'

# 使用
ibdashboard
```

## 💡 使用技巧

### 快速查看数据
```bash
# 查看最新净值
python3 -c "import json; d=json.load(open('data/dashboard_data.json')); print(f'净值：¥{d[\"summary\"][\"totalNav\"]:,.0f}')"

# 查看前 10 大持仓
python3 -c "
import json
d=json.load(open('data/dashboard_data.json'))
positions = d['openPositions']['stocks'] + d['openPositions']['etfs']
for p in sorted(positions, key=lambda x: x['positionValue'], reverse=True)[:10]:
    print(f'{p[\"symbol\"]:10} ${p[\"positionValue\"]:>12,.0f}')
"
```
